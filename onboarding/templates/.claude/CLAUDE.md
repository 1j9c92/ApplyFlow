# ApplyFlow Career Context Router

This file points to the career context modules. Use it to navigate and load the right knowledge base depending on the task.

---

## Career Context Modules

Load these files when relevant to your task:

### Core Career Context

- **`career-context/profile.md`** — Your professional identity, contact info, career narrative, positioning
- **`career-context/preferences.md`** — Application defaults, writing voice rules, resume rules
- **`career-context/education.md`** — Formal education, certifications, bootcamps, learning
- **`career-context/skills-and-tools.md`** — Technical, business, and interpersonal skills organized by category
- **`career-context/work-history/`** — Individual entries for each role (most recent first)
- **`career-context/job-search-criteria.md`** — Target roles, industries, company stage, geography, compensation, work environment
- **`career-context/resume-bullets.md`** — Pre-approved accomplishment statements tagged with skills

### Writing & Voice

- **`cover-letter-loop/write-like-user.md`** — Your authentic writing voice and style rules

---

## Domain Files

Load these to capture lessons learned over time:

- **`domains/job-search.md`** — Job search strategy, what works, application patterns, learnings
- **`domains/ai-automation.md`** — How you use AI and automation, what works, what does not
- **`domains/productivity.md`** — Your task and project management approach
- **`domains/personal.md`** — Personal finance, life admin, long-term planning

---

## How to Use This Router

1. **At the start of a session**: Check which task you are working on (e.g., tailoring a resume, writing a cover letter, applying to a role, tracking what works)
2. **Load the relevant modules**: Use this router to identify which files to read
3. **Update domain files after sessions**: When you learn something useful, update the relevant domain file

### Task → Module Map

| Task | Load |
|------|------|
| Writing a cover letter | profile.md, preferences.md, job-search-criteria.md, resume-bullets.md, write-like-user.md |
| Tailoring a resume | profile.md, work-history/, resume-bullets.md, skills-and-tools.md |
| Prepping for an interview | profile.md, work-history/, resume-bullets.md, job-search-criteria.md |
| Evaluating a job opportunity | job-search-criteria.md, job-search.md domain |
| Building a job search strategy | job-search-criteria.md, job-search.md domain, preferences.md |
| Reflecting on application results | job-search.md domain, ai-automation.md domain |
